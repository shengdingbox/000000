# 游戏功能说明 (Game Features)

## 新增游戏场景元素

游戏现在包含完整的可玩游戏场景！

### 玩家飞船 (Player Ship)
- **外观**: 绿色三角形，白色边框
- **位置**: 屏幕底部中央
- **控制**:
  - 左右移动: 方向键 ← → 或 A/D 键
  - 发射子弹: 空格键 或 W 键
- **属性**:
  - 生命值: 3 条
  - 射击速度: 每250毫秒一次
  - 移动速度: 5 像素/帧

### 敌人 (Enemies)
- **外观**: 红色倒三角形，粉色边框
- **行为**: 从屏幕顶部随机位置出现，向下移动
- **生成频率**: 每1.5秒生成一个新敌人
- **速度**: 2-4 像素/帧（随机变化）
- **碰撞**: 与玩家或子弹碰撞后消失

### 子弹 (Bullets)
- **外观**: 黄色矩形
- **速度**: 7 像素/帧
- **行为**: 向上飞行，击中敌人后消失

### 游戏机制

#### 得分系统
- 每击毁一个敌人获得 10 分
- 分数显示在左上角

#### 生命值系统
- 玩家初始有 3 条生命
- 与敌人碰撞失去 1 条生命
- 生命值显示在分数下方
- 生命值为 0 时游戏结束

#### 碰撞检测
- 玩家子弹击中敌人 → 敌人消失，得 10 分
- 敌人碰到玩家 → 敌人消失，玩家失去 1 条生命

#### 游戏结束
- 当生命值降为 0 时触发
- 显示 GAME OVER 画面
- 显示最终分数
- 提示刷新页面重新开始

### 调试信息 (DEBUG模式)
当 `CONFIG.DEBUG = true` 时，显示：
- FPS (帧率)
- 当前敌人数量
- 当前子弹数量

### 背景
- 动态星空背景
- 蓝色渐变（深蓝到浅蓝）
- 50 个移动的白色星星

## 技术实现

### 类结构
- `Player` - 玩家飞船类
- `Enemy` - 敌人类
- `Bullet` - 子弹类
- `Game` - 主游戏类（管理所有实体）

### 游戏循环
1. 更新玩家位置
2. 处理射击输入
3. 生成新敌人
4. 更新所有敌人位置
5. 更新所有子弹位置
6. 检测碰撞
7. 清理超出边界的实体
8. 渲染所有元素

### 性能
- 使用 `requestAnimationFrame` 实现流畅的 60 FPS
- 自动清理屏幕外的对象避免内存泄漏
- 高效的碰撞检测算法

## 游戏玩法
1. 点击 "Start Game" 按钮开始
2. 使用方向键或 A/D 移动飞船
3. 按空格键或 W 键发射子弹
4. 击毁敌人获得分数
5. 避免与敌人碰撞
6. 尽可能获得高分！
