# 游戏视觉指南 (Visual Guide)

## 游戏界面布局

```
┌─────────────────────────────────────────────────────────┐
│  Score: 0                Lives: 3                        │
│                                                           │
│                                                           │
│                        ▼ ▼ ▼                             │
│                     (红色敌人)                           │
│                                                           │
│                          │                                │
│                          │ (黄色子弹)                    │
│                          │                                │
│                                                           │
│                                                           │
│                          ▲                                │
│                      (绿色玩家)                          │
│                                                           │
└─────────────────────────────────────────────────────────┘
```

## 游戏元素详解

### 1. 玩家飞船 (Player Ship)
```
      ▲
     / \
    /   \
   /_____\
```
- **颜色**: 亮绿色 (#00ff88)
- **边框**: 白色 (2px)
- **形状**: 指向上方的三角形
- **位置**: 屏幕底部中央，距底部20像素
- **尺寸**: 40x40 像素
- **控制**: 
  - 键盘方向键 ← → 
  - 或 A/D 键

### 2. 敌人飞船 (Enemy Ships)
```
   _____
   \   /
    \ /
     ▼
```
- **颜色**: 红色 (#ff4444)
- **边框**: 粉红色 (#ff8888, 2px)
- **形状**: 指向下方的三角形
- **生成**: 从屏幕顶部随机X坐标
- **尺寸**: 35x35 像素
- **速度**: 2-4 像素/帧（随机）
- **生成频率**: 每1.5秒一个

### 3. 子弹 (Bullets)
```
   |
   |
```
- **颜色**: 黄色 (#ffff00)
- **形状**: 矩形
- **尺寸**: 4x12 像素
- **速度**: 7 像素/帧
- **方向**: 向上
- **发射**: 空格键或W键
- **冷却**: 250毫秒

### 4. 背景 (Background)
- **颜色**: 深蓝到浅蓝的渐变 (#000428 → #004e92)
- **星星**: 50个白色小圆点
- **动画**: 星星缓慢向左移动，营造太空感

### 5. UI元素

#### 左上角信息
```
Score: 150      ← 当前分数
Lives: 2        ← 剩余生命值
```

#### 调试信息 (DEBUG模式)
```
FPS: 60         ← 当前帧率
Enemies: 3      ← 屏幕上的敌人数量
Bullets: 2      ← 屏幕上的子弹数量
```

#### 游戏结束界面
```
┌─────────────────────────────────┐
│                                 │
│          GAME OVER              │
│                                 │
│      Final Score: 150           │
│                                 │
│    Refresh to Play Again        │
│                                 │
└─────────────────────────────────┘
```
- **背景**: 半透明黑色遮罩
- **标题**: 大号红色字体
- **分数**: 白色字体
- **提示**: 白色字体

## 游戏流程

### 开始界面
```
┌─────────────────────────────────┐
│                                 │
│       Space Shooter             │
│                                 │
│      ┌─────────────┐           │
│      │ Start Game  │           │
│      └─────────────┘           │
│                                 │
└─────────────────────────────────┘
```

### 游戏进行中
1. 玩家在底部，可以左右移动
2. 敌人从顶部持续生成
3. 按空格或W发射子弹
4. 子弹击中敌人，敌人消失，得分+10
5. 敌人碰到玩家，失去1条生命
6. 生命值为0时游戏结束

## 碰撞效果

### 子弹击中敌人
```
Before:         After:
  ▼              (消失)
  │
  ▲
```
- 敌人消失
- 子弹消失
- 分数 +10

### 敌人撞到玩家
```
Before:         After:
  ▼              (敌人消失)
  ▲              ▲ (生命-1)
```
- 敌人消失
- 生命值 -1
- 玩家保持在原位

## 颜色方案

| 元素 | 主色 | 辅助色 |
|------|------|--------|
| 玩家 | #00ff88 (亮绿) | #ffffff (白色边框) |
| 敌人 | #ff4444 (红色) | #ff8888 (粉红边框) |
| 子弹 | #ffff00 (黄色) | - |
| 背景 | #000428 → #004e92 (蓝色渐变) | #ffffff (星星) |
| UI文字 | #ffffff (白色) | - |
| 调试信息 | #00ff88 (绿色) | - |
| 游戏结束 | #ff4444 (红色标题) | #ffffff (其他文字) |

## 动画效果

1. **星空背景**: 持续水平移动
2. **敌人**: 垂直向下移动
3. **子弹**: 快速向上移动
4. **玩家**: 响应键盘输入平滑移动

## 响应式设计

- Canvas自动适应窗口大小
- 保持 800:600 的宽高比
- 最大占用窗口90%的空间
